# Netflix Content Analytics

## Overview
This project analyzes Netflix's global content library using SQL, Microsoft Excel, and Power BI to identify trends in genres, content ratings, release years, and regional distribution. The objective was to transform raw data into meaningful business insights through data analysis and interactive visualizations.

## Objectives
- Analyze Netflix content trends across different genres and countries.
- Study content distribution based on ratings and release years.
- Build an interactive dashboard for business insights.
- Practice SQL querying and Power BI dashboard development.

## Tools & Technologies
- SQL
- Microsoft Excel
- Power BI

## Methodology
- Cleaned and organized the dataset.
- Used SQL queries to analyze content trends.
- Performed exploratory data analysis.
- Developed an interactive Power BI dashboard with multiple visualizations.
- Interpreted results to identify audience and content trends.

## Key Insights
- Drama and International Movies were among the most common content categories.
- Significant growth in Netflix content was observed after 2016.
- The United States contributed the largest share of content.
- Content ratings varied across different regions and genres.
